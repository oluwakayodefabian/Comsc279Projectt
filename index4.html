Introduction:

CPU scheduling is the process of managing how processes share CPU resources in multitasking systems.It is also a process used by the operating system to decide which task or program gets to use the CPU at a particular time.

Main Function of CPU Scheduling
Is to ensure that whenever the CPU remains idle, the OS has at least selected one of the processes available in the ready-to-use line

Types of CPU Scheduling

  1. Preemptive Scheduling: is used when a process switches from running state to ready state or from the waiting state to the ready state. Algorithms under this include;
    • Priority-Scheduling
    • Round Robin

  2. Non-Preemptive Scheduling: This is the opposite of preemptive scheduling as it is used when a process terminates , or when a process switches from running state to waiting state. Algorithms under this include:
    • FCFS - First Come First Serve
    • SJF - Shortest Job First
This guide explores various CPU scheduling algorithms, including their pros, cons, and use cases.
